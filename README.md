# Calendar Sidebar — Obsidian Plugin / Obsidian 插件

DayOne-style monthly calendar panel in the Obsidian left sidebar. / DayOne 风格的月历面板，显示在 Obsidian 左侧侧边栏。

---

## Features / 功能

- **Monthly calendar** in the left sidebar / 月历视图显示在左侧侧边栏
- **Image thumbnails** from daily notes as date cell backgrounds / 自动提取日记图片作为日期格子缩略图
- **Today highlight** (full accent fill) + **Browsing-date highlight** (accent border) / 今日高亮（全色块）+ 浏览中日期高亮（彩色边框）
- **One-click open** daily notes / 单击一键打开日记
- **Auto-create missing notes** with confirmation dialog, from Daily Notes template (Templater supported) / 点击无日记日期 → 确认框 → 从模板自动创建（支持 Templater）
- **Configurable daily folder** with search suggest / 可配置日记文件夹路径（支持搜索）
- **Thumbnail filter**: all images or date-prefixed only / 缩略图过滤：全部图片或仅日期前缀图片

## Installation / 安装

- **BRAT**: Add `QVQSmile/obsidian-calendar-sidebar` to BRAT / 在 BRAT 中添加 `QVQSmile/obsidian-calendar-sidebar`
- **Manual**: Download `calendar-sidebar.zip` from [Releases](https://github.com/QVQSmile/obsidian-calendar-sidebar/releases), extract to `.obsidian/plugins/calendar-sidebar/` / 从 Releases 下载 `calendar-sidebar.zip`，解压到 vault 的 `.obsidian/plugins/calendar-sidebar/`

## Requirements / 要求

- Obsidian v1.5.0+
- Daily notes named `YYYY-MM-DD.md` / 日记按 `YYYY-MM-DD.md` 命名
- Images embedded via `![[image.jpg]]` / 图片通过 `![[image.jpg]]` 嵌入
